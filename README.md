# 현승준 | 프론트엔드 개발자 — 포트폴리오

탐구한 흔적을 기록하고 구조화된 결과물로 이어내는 개인 포트폴리오 웹사이트입니다.

- **Tech**: React, Vite, i18n (한국어/영어)
- **배포**: [seungjoonh.github.io](https://seungjoonh.github.io/)

<br />

## 사용 방법

### 설정 (Settings)

우측 상단 **설정(톱니바퀴)** 버튼으로 열 수 있습니다.

| 항목 | 설명 |
|------|------|
| **전체 속도** | 스크롤 / 애니메이션 속도 (0.5x ~ 1.5x) |
| **글자 크기** | 전역 폰트 스케일 (0.5x ~ 1.5x) |
| **언어** | 한국어 / English 전환 |
| **테마** | 라이트 / 다크 모드 전환 |

설정은 브라우저 로컬 스토리지에 저장되어 다음 방문 시 유지됩니다.

<br />

### 프로젝트 검색 (Search)

프로젝트 섹션 상단 **검색창**에서 키워드·접두어로 프로젝트를 필터링할 수 있습니다.

- **한글**: 초성·자모·전체 단어 검색 지원
- **여러 조건**: `|` 로 OR, 공백/쉼표로 AND

#### 검색 접두어

| 접두어 | 설명 | 예시 |
|--------|------|------|
| `title:` | 프로젝트 제목 | `title:물방울톡`, `title:"Moolbangwool Talk"` |
| `desc:` | 요약·섹션 본문 | `desc:"컴포넌트 설계"` |
| `stack:` | 기술 스택 | `stack:React`, `stack:TS` |
| `tag:` | 태그 | `tag:유지보수` |
| `#태그` | 태그 (짧은 형식) | `#바이브코딩`, `#물방울톡` |
| `type:` | 프로젝트 유형 | `type:group`, `type:personal`, `type:toy` |
| `team:` | 팀 인원 | `team:4` |
| `link:` | 링크 유형 | `link:github` |
| `show:` | 표시 범위 | `show:public`, `show:hidden`, `show:all` |

- **따옴표** `"..."`: 공백 포함 시 정확히 일치
- **OR**: `title:Fitween \| title:Extify` — 둘 중 하나라도 만족하면 표시
- **숨김 프로젝트**: `show:all` 로 함께 검색·표시 가능

#### 검색 단축 적용

- **경력 카드**의 "프로젝트 바로가기" 클릭 → 해당 프로젝트 검색어가 검색창에 적용
- **프로젝트 상세**의 태그/스택 클릭 → 해당 태그·스택으로 검색
- **스킬** 섹션의 스킬 클릭 → 해당 스택으로 프로젝트 검색
- **문서(Docs)** 칩 클릭 → 관련 프로젝트 검색어 적용
